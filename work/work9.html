Weather Tracker App

Description

A web based weather tracker using openweathermap api to get the accurate location and temperature. This is a personal project inspired by other developer. You can input the location you want to know the temperature of and the background will change depending on the hotness or the coldness of the location temperature.

Features:
  • This Website can track location temperature
  • This Website can change depending on the location's temperature.
  • This Website is responsive.
