Cebu Home Builders E-Catalog

Description

A web based catalogue containing the details and pictures of items currently being offered for sale and used by the company to show to the consumers. It is completely responsive from desktop to tablets and mobile. It is in a form of a book or magazine but in a digital form of way. We used an extension that converts pdf to a flipbook.

Features:
  • This Website can display products in a e-catalogue style.
  • This Website also displays video and can continously play until it is stop by the user.
  • This Website is responsive.
  • This Website can search any catalogue containing the input keywords.
  • This Website has a Tile calculator.
