ITech E-shop Multi-Vendor Ecommerce

Description

This is a multi-vendor ecommerce built from scratch and not using any cms tools. It is pure coded website. I propose this project so local small business would not be overshadowed by big business overseas. It is available only in that certain area since it has a confirmation feature so the admin has the full control to only local business can register in the website.

Features:
  • This Website has a customer and admin dashboard
  • This Website can cater customer orders and track the status of the order.
  • This Website can accept and cancel order from customers.
  • This Website can register unlimited users and sellers.
  • This Website has a main admin daashboard that can track users and sellers.
  • This Website has an appointment function.
  • This Website can generate sales report.
  • This Website can send email regarding appointment and order status.
  • This Website can rate stars and review from customers to each product after order completed

  • This website is not fully responsive and production is stopped due to certain circumstances.
